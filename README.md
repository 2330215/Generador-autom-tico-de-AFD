# Generador-automatico-de-AFD
proyecto final de Lenguajes y Automatas
Proyecto: Herramienta Gráfica para ER → AFND → AFD y Minimización

Este proyecto implementa una  serie de herramientas gráficas en Python para trabajar con expresiones regulares, autómatas y minimización de AFD, utilizando pygame para la interfaz visual.

Incluye:

Parser y validador de expresiones regulares

Conversión de ER a AFND

Conversión de AFND a AFD

Minimización de AFD

Menú principal para navegar entre las herramientas
